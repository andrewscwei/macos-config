# ZSH

This directory contains personal ZSH configurations.

## Importing Config

Install via cURL:

```sh
$ curl -o- -s https://raw.githubusercontent.com/andrewscwei/macos-config/master/zsh/install.sh | bash
```

Install locally:

```sh
$ ./install.sh
```

## Exporting Config

```sh
$ ./export.sh
```
