## Usage

### Installing Config

Install via cURL:

```sh
$ curl -o- -s https://raw.githubusercontent.com/andrewscwei/vscode-config/macos/install.sh | bash
```

### Exporting Existing Config

```sh
$ git clone --single-branch --branch macos https://github.com/andrewscwei/vscode-config
$ cd vscode-config
$ ./export.sh
```
